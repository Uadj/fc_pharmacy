# fastcampus 약국 찾기 서비스 강의 클론

## 실행방법

JAR 빌드

``` ./gradlew clean build -PKAKAO_REST_API_KEY= 본인의 카카오 REST API키 ```

컨테이너 실행

``` docker-compose -up ```
